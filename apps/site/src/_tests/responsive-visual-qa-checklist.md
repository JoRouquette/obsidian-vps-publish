# Responsive Visual QA Checklist

## Mobile 360px to 430px

- Viewer: text is readable without zoom and keeps visible lateral padding.
- Viewer: tables and code blocks scroll inside their wrapper without widening the page.
- Topbar: compact mobile breadcrumb is visible without horizontal page scrolling.
- Explorer drawer: opens without horizontal overflow and keeps touch-friendly rows.
- Leaflet blocks: map remains bounded in the reading column and page scroll stays fluid around it.

## Tablet 768px

- Viewer: reading width remains comfortable and headings still separate sections clearly.
- Topbar: breadcrumbs stay understandable and title/actions stay aligned.
- Explorer: drawer/panel width feels stable and long labels still truncate cleanly.

## Desktop 1280px+

- Viewer: reading measure is still constrained and not overly wide.
- Breadcrumbs: desktop breadcrumb remains fully visible and unchanged in hierarchy.
- Overlays and drawers: rounded panel treatment remains intact with no visual jump from mobile rules.
